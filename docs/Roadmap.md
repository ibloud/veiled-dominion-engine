# Roadmap

Planned milestones and features for the Veiled Dominion prototype.

Near-term
- Basic playable prototype with core movement and captures
- Restraint scoring implementation
- Single-board multiplayer UI

Mid-term
- AI opponents (basic heuristics)
- Playtesting harness and automated simulation
- Visual polish (SVG board, piece assets)

Long-term
- Multiplayer lobby & networking
- Variant ruleset management (load/save variants)
- Tutorial and demo scenarios

If you want, I can convert this into a project board with GitHub Projects or add issues for each roadmap item.
