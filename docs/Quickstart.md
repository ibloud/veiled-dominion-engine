# Quickstart

This page helps you get the prototype running locally or play a demo if one is available.

Prerequisites
- Git
- A modern web browser (Chrome, Firefox, Safari)
- Node.js (if the prototype is a web app) — version X.X or newer (replace with actual version if known)

Run locally (web prototype)
1. Clone the main repo:
   `git clone https://github.com/ibloud/veiled-dominion-engine.git`
2. Enter the project:
   `cd veiled-dominion-engine`
3. Install dependencies:
   `npm install`
4. Run the dev server:
   `npm run dev`
5. Open your browser to `http://localhost:3000` (or the port from the console)

Play demo (no local run)
- If a hosted demo URL exists, it will be linked here (add demo URL).
- Alternatively, open `index.html` from the repo in a browser if the prototype is static.

Troubleshooting
- Build errors: check Node version and run `npm ci` to get a clean install.
- No UI: check the browser console for errors and report them in Issues.

If you don’t have the prototype runnable locally and want a short video/GIF for the docs, I can prepare one if you supply a built demo or host the build temporarily.
